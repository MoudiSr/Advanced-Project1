# My Restaurant Project

A simple restaurant web application where users can browse the menu, add items to the cart, place orders, and track them. Admins can manage kitchen orders.

## Features
- Login Page for customer and Admin
- Home page with image carousel and introduction
- Menu page with dishes, drinks, and desserts
- Cart system with dynamic total price
- Order tracking for customers
- Kitchen dashboard for admin to update order status
- About page with chef team
- Contact form with message submission
- Responsive design using Bootstrap
